# Assets

This directory is a placeholder.  The current prototype renders all
objects with built-in Godot primitives (SphereMesh, CylinderMesh,
SurfaceTool lines) and procedural materials — no binary art assets are
checked in by design.

When higher-fidelity assets are introduced (textures, glTF meshes,
particle systems, sky panoramas, etc.), they belong here, organised by
category:

```
assets/
  meshes/
  materials/
  textures/
  particles/
  audio/
```

Keep individual files small and prefer text-based formats (.svg, .obj,
.gltf) over large binaries when possible.
