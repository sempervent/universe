# Unreal data bundle (optional)

Generated by:

```bash
uv run universe game export-unreal-data \
  --scene data/generated/scene-001/scene.json \
  --out frontends/unreal/Data
```

Files:

- `scene_unreal.json` — convenience transform (render positions + hints)
- `featured_objects.json` — camera / teaching targets
- `material_hints.json` — per-type material profiles
- `manifest.json` — file list

Canonical source remains `data/generated/scene-001/scene.json`.

Set `AUniverseSceneActor::bPreferUnrealBundle` and point `SceneJsonPath` at `scene_unreal.json` to use the bundle.
